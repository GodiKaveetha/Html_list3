What is Internet?

Internet used to communicate between user and system. Ex: Sending Emails, Online Shopping.

What is Browser?

A Browser is a software application used to access and view websites on the internet. Ex: Google Chrome, Firefox, Safari.

What is URL?

URL stands for Uniform Resource Locator. It is the web address used to access resources like a specific web page, image, video or a file.

What is HTTP and HTTPS?

HTTP stands for HyperText Transfer Protocol and HTTPS stands for HyperText Transfer Protocol Secure. HTTP and HTTPS both are protocols used to transfer data over the web.

Difference between HTTP and HTTPS?

HTTPS encrypts the data for security.

HTTP sends data without encryption.

What is IP Address?

It is a unique number and it identifies a device on the internet

What is Syntax?

It is the set of rules that tells how to write code correctly in a programming language.

What is Domain?

Domain refers to the address or location of a website or a web resource.

Ex: www.amazon.com

What is debug?

Debug is the process of finding and fixings bugs or errorsin a software program.

What is markup language?

A markup language is a system of codes or tags used to define the structure, formatting or presentation of text in document.

What is programming language?

A programming language is a formal language used to give instructions to a computer to perform specific tasks or operations.
